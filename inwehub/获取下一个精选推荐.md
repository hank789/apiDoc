# 获取下一个精选推荐

## 接口地址

`/api/recommendRead/getNext`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| recommend_id   | integer   | yes     | 当前推荐的id |



## 返回体

如果没有下一个，返回数据为空
