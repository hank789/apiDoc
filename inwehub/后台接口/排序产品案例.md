# 排序产品案例

## 接口地址

`/api/manage/product/sortCase`

## 请求方式

`post`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| case_id  | integer   | yes      | 案例id  |
| to_case_id   | string   | yes    | 交换排序的案例id  |

## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
      "id": 13,//案例id
      "sort": 4,//新的排序
  }
}
```
