# 显示|隐藏|删除资讯

## 接口地址

`/api/manage/product/updateNewsStatus`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| news_id   | integer   | yes     | 文章id |
| status   | integer   | yes     | 状态：0隐藏，1显示，3删除 |



## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": [
    "status": 1,//当前状态：0隐藏，1显示，3删除
  ]
}
```
