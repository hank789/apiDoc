# 添加内容源

## 接口地址

`/api/manage/product/storeSource`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | integer   | yes     | 产品id |
| source   | string   | yes     | source源 |


## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": [
    "id": 13,//源id
  ]
}
```
