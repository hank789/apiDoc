# 获取源信息

## 接口地址

`/api/manage/product/fetchSourceInfo`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| product_id   | integer   | yes     | 产品id |
| source   | string   | yes     | source源 |


## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": [
    "title": "主题",//标题
  ]
}
```
