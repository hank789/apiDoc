# 点评列表

## 接口地址

`/api/tags/productReviewList`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| tag_name   | string   | yes     | 产品名字 |
| page   | string   | no      | 页数，默认1 |
| perPage   | string   | no      | 每页条数，默认20 |
