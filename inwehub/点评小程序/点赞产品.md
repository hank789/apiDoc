# 点赞产品

## 接口地址

`/api/weapp/product/supportAlbumProduct`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | integer   | no     | 产品id |

## 返回体

```json5
{
  "status": true,
  "code": 1000,//如果返回6120，说明当天点赞次数已用完
  "message": "操作成功",
  "data": {
      "id": "1",//专辑id
      "name": "SAP",            
      "icon": 'http://www.test.com/img.png',
      "summary": "简介",
     
  }
}
```
