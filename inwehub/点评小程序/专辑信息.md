# 专辑信息

## 接口地址

`/api/weapp/product/albumInfo`

## 请求方式

`get`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | integer   | no     | 专辑id |

## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
      "id": "1",//专辑id
      "name": "SAP",            
      "icon": 'http://www.test.com/img.png',
      "summary": "简介",
     
  }
}
```
