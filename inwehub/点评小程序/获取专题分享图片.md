# 获取专题分享图片

## 接口地址

`/api/weapp/product/getAlbumShareImage`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | integer   | yes     | 专辑id |



## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": [
    "url": "https://cdn.inwehub.com/demand/qrcode/2018/11/1542631958deJB10z.png"
  ]
}
```
