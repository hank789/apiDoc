# 完善手机号

## 接口地址

`/api/weapp/user/updatePhone`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| phone   | string   | yes      | 手机号 |
